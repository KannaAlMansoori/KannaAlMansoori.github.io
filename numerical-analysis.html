Numerical Analysis

Period: April 2025 A | Status: In Progress

This module introduces key aspects of mathematics and statistics for data science and artificial intelligence. The central consideration of this module is the need to be aware of, and be able to critically evaluate, the range of mathematical and statistical methodology seen in both disciplines, and to build a foundation for analysing and solving problems in both disciplines. All the statistical techniques and data analysis, as well as assessments, will be performed using R programme.

Reading in this Module

The core eBook for this module is Bader, M. and Leuzinger, S. (2024) R-ticulate: A Beginner's Guide to Data Analysis for Natural Scientists. 1st edn. Hoboken: Wiley.

R-ticulate: A Beginner's Guide to Data Analysis for Natural Scientists is a compact, example-based, and user-friendly statistics textbook with a unique content structure that introduces statistical skills simultaneously with software (programming) skills in R, providing an integrated approach to learning both theoretical concepts and practical application tools essential for data analysis.

The core eBook can be accessed via the University of Essex Library from the eBook tab.

In addition to the core eBook, this module has a reading list which can be found within Module Resources that is also available via the University of Essex Reading List service (Talis). Each unit in this module also has a unit reading page.

Module Aims and Learning Outcomes

In this module, we shall acquire:

  • Develop a systematic understanding of foundational mathematical principles and methods, as well as core and specialised concepts underpinning computing logic.
  • Understand the foundation for the development and application of programming and data-driven techniques, from both a theoretical and practical viewpoint.
  • Facilitate the ability to interpret the results generated when using these data science and AI tools.
  • Gain an understanding of the real-world applications of these computational tools, and contemporary issues related to these computational techniques.
  • The opportunity to take a reflective and independent approach to the learning process.
