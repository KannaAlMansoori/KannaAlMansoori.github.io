Frequency Analysis of COVID-19 Recovery Cases in India

January 2020 - March 2020

üéØ Introduction

This data activity focuses on analyzing the COVID-19 India Dataset covering the early pandemic period from January 2020 to March 2020. The primary objective is to perform frequency analysis to understand patterns in recovery reporting across Indian states during the initial outbreak phase.

Using statistical methods in R, we examine how consistently recovery cases were reported during this critical period, providing insights into the early pandemic response and data collection practices.

üìã Task Requirements

Objectives:
  • Create a binary variable (has_recovery) that indicates whether any recoveries (cured cases) were reported
  • Generate a frequency table to count daily reports with recoveries versus no recoveries using the table() command
  • Calculate percentages to understand the proportion of daily state reports that included recovery cases
  • Analyze and interpret the findings in the context of early pandemic reporting patterns

üî¨ Methodology

Data Preparation

The analysis was conducted using R programming language. The dataset was loaded and prepared for frequency analysis through the following steps:

# Load the required library library(readr) # Load the COVID-19 India dataset Covid19_India_Jan_20_Mar_20_ <- read_csv("Numercial computer M3/Covid19 India (Jan 20 - Mar 20).csv") View(Covid19_India_Jan_20_Mar_20_) # Rename the dataset for easier handling df <- Covid19_India_Jan_20_Mar_20_

Step 1: Create a Binary Variable

A binary variable has_recovery was created to classify each daily report as either having recoveries ("Yes") or no recoveries ("No"):

# Create binary variable for recovery status df$has_recovery <- ifelse(df$Cured > 0, "Yes", "No")

Step 2: Create a Frequency Table

Using the table() function to generate a frequency distribution:

# Generate frequency table recovery_table <- table(df$has_recovery) recovery_table

Step 3: Calculate Percentages

Converting frequencies to percentages using prop.table():

# Calculate percentage distribution recovery_percentage <- prop.table(recovery_table) * 100 recovery_percentage

üîç Analysis and Findings

Key Findings:
  • Majority of reports showed no recoveries: Out of 270 daily state reports, 215 (79.63%) reported zero recovery cases during this early pandemic period.
  • Limited recovery reporting: Only 55 reports (20.37%) documented at least one recovery case, indicating that recoveries were relatively rare or inconsistently reported in the initial months.
  • Early pandemic dynamics: The data reflects the early stage of the pandemic in India (January-March 2020) when:
    • The outbreak was just beginning to spread
    • Case numbers were still relatively low
    • Recovery times meant that many patients hadn't yet recovered
    • Data collection and reporting systems were still being established
  • Statistical insight: The approximately 4:1 ratio (no recoveries to recoveries) suggests that during this initial period, the focus was primarily on tracking new cases and the disease was spreading faster than patients were recovering.
  • Data quality considerations: The binary classification helps identify patterns in reporting consistency and can highlight potential gaps in early pandemic surveillance systems.
