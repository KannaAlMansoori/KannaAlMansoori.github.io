Unit 2 Seminar Preparation

Introducing First Order Logic

Type: Seminar Preparation | Status: ✓ Complete

Seminar Overview

Reference: Russell and Norvig (2021) - Chapter 8: Knowledge Representation and Reasoning

This seminar introduces First Order Logic (FOL), an extension of propositional logic that allows us to represent knowledge about individual objects, their properties, and relationships. FOL is fundamental to AI knowledge representation and enables more expressive and scalable reasoning systems.

The 4 Main Building Blocks of First Order Logic

1. Constants

Specific objects or names that refer to definite things in the world. Examples: John, Mary, 5, Paris.

2. Variables

Placeholders or blank spaces representing unspecified objects. Examples: x, y, z. Variables allow us to write general statements that apply to any object.

3. Predicates

Describe properties of objects or relationships between objects. Examples: "is mortal," "loves," "is tall," "is a student." Used in statements like "John is tall" or "Mary loves John."

4. Functions

Connect objects to other objects. Examples: "the father of," "the age of," "the mother of." So "the father of John" would give us another object.

The Two Quantifiers (Most Important!)

The Universal Quantifier: "For All" (∀)

Expresses that something is true about ALL objects in a group.

Example: "All men are mortal"

In First Order Logic: "For every person x, if x is a man, then x is mortal."

Other examples:

  • All students are intelligent
  • Every dog is an animal
  • All cats have four legs

The Existential Quantifier: "There Exists" (∃)

Expresses that something is true about AT LEAST ONE object.

Example: "Some people like ice cream" or "There exists someone who likes ice cream"

In First Order Logic: "There is at least one person x who likes ice cream."

Other examples:

  • There is a student who failed the exam
  • Someone is the richest person in the world
  • There exists a white alligator

The Classic Example: Deductive Reasoning

This example is fundamental to understanding how First Order Logic enables reasoning:

Statement 1: Every person is mortal

Statement 2: Confucius is a person

Conclusion: Therefore, Confucius is mortal

This demonstrates how First Order Logic allows us to make general statements (all persons are mortal) and apply them to specific individuals (Confucius). This is the power of FOL over propositional logic.

Converting English to Logic Patterns

Pattern 1: "All X are Y"

  • All students are intelligent
  • All dogs are animals
  • All men are mortal

Pattern 2: "Some X are Y" or "There is/are"

  • Some students are intelligent
  • There is a dog in the park
  • Some people like coffee

Pattern 3: "Everyone loves someone"

Means each person loves at least one other person (∀x ∃y Loves(x, y))

Pattern 4: "Someone loves everyone"

Means there is one person who loves all other people (∃x ∀y Loves(x, y))

Why First Order Logic is More Powerful

With basic propositional logic, if you wanted to say "pits cause breezes in adjacent squares," you'd have to write one sentence for EVERY square in a game. That's hundreds or thousands of sentences!

With First Order Logic, you can write ONE general rule using quantifiers, and it applies to all squares automatically. This scalability and expressiveness is why FOL is fundamental to AI knowledge representation.

What to Be Ready for in the Seminar

  • Translate English to Logic: Convert normal sentences into logical form
  • Choose the Right Quantifier: Know when to use "for all" versus "there exists"
  • Identify Components: Distinguish between constants, variables, predicates, and functions
  • Apply to New Examples: Use FOL concepts to represent knowledge about unfamiliar scenarios
  • Reason Deductively: Work through logical arguments and draw conclusions

Quick Review Checklist

  • ✓ Constants = specific objects (John, 5, Paris)
  • ✓ Variables = placeholders (x, y, z)
  • ✓ Predicates = properties or relationships (is tall, loves, is a student)
  • ✓ Functions = connections between objects (father of, age of)
  • ✓ For All (∀) = something is true about EVERY member of a group
  • ✓ There Exists (∃) = something is true about AT LEAST ONE member of a group

Key Takeaways

  • First Order Logic extends propositional logic with variables, predicates, and quantifiers
  • Quantifiers (for all, there exists) are the most crucial concepts for expressing general knowledge
  • FOL enables scalable knowledge representation—one rule can apply to many objects
  • Converting English to logical form is a core skill for reasoning systems
  • Understanding the difference between universal and existential statements is fundamental
