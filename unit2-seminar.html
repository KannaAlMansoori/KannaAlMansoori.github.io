Seminar Questions and My Answers

Question 1: Primary Purpose and Differences

Question: What is the primary purpose of descriptive statistics, and how does it differ from other statistical methods?

My Answer:

Descriptive statistics organizes and summarizes data to make it easier to understand. According to Dong (2023), it converts scattered observations into meaningful summaries like means, medians, and standard deviations. Unlike inferential statistics, which makes predictions about larger populations, descriptive statistics simply describes what's actually in your dataset—it's the foundation for more complex analyses.

Question 2: Beijing Housing Prices - Key Factors (Table 2)

Question: According to the paper's analysis of housing prices in Beijing (Table 2), what were the key factors that contributed to the increase in housing prices between 2012 and 2015? Explain how the descriptive statistics helped identify these factors.

My Answer:

According to Table 2, the key factors contributing to Beijing's housing price increase from 2012 to 2015 were property size characteristics: living space grew from 102.52 to 123.94 square meters, average bedrooms increased from 2.126 to 2.415, and building height rose from 14.099 to 15.342 floors. These changes aligned with the overall price jump from 336.727 to 477.521.

Descriptive statistics helped identify these factors by enabling year-to-year comparisons through means and standard deviations. This made it easy to spot which characteristics changed significantly alongside prices. Importantly, the statistics also showed that orientation remained stable, indicating it wasn't a contributing factor—demonstrating how descriptive statistics can both confirm and eliminate potential explanatory variables.

Question 3: Mean, Median, and Mode - Calculation

Question: Explain the difference between mean, median, and mode. Given the sample: 13, 15, 17, 15, 20, 15, 18, calculate all three measures of central tendency.

My Answer:

The three measures of central tendency differ in how they represent typical values: the mean is the arithmetic average (sum ÷ count), the median is the middle value when data is ordered, and the mode is the most frequently occurring value.

Calculations for the sample:
Given: 13, 15, 17, 15, 20, 15, 18
Ordered: 13, 15, 15, 15, 17, 18, 20

Mean = (13 + 15 + 17 + 15 + 20 + 15 + 18) ÷ 7 = 113 ÷ 7 = 16.14
Median = Middle value (4th position) = 15
Mode = Most frequent value (appears 3 times) = 15

Question 4: Main Limitations

Question: What are the main limitations of descriptive statistics as identified in the paper? Why is descriptive statistics alone not sufficient for comprehensive statistical analysis?

My Answer:

According to Dong (2023), descriptive statistics has three main limitations. First, it cannot predict future trends—it only describes existing data and compares what already exists, without forecasting capabilities. Second, data collection can be time-intensive, as demonstrated by the stock market example where gathering comprehensive data took researchers months. Third, and most importantly, descriptive statistics alone is insufficient because it summarizes data without explaining causation or relationships between variables.

This is why descriptive statistics serves as a starting point rather than a complete solution. It provides the foundation for analysis but must be combined with inferential statistics or regression models for comprehensive understanding.

Question 5: Stock Market and Trust Relationship (Tables 3 & 4)

Question: Based on Table 3 and Table 4 in the stock market analysis section, explain how descriptive statistics (particularly mean values) helped researchers determine the relationship between trust levels and stock market volatility during COVID-19.

My Answer:

Descriptive statistics revealed the trust-volatility relationship through systematic comparison of mean values. Table 3 established baseline measures for social trust, government trust, and market volatility. Table 4 then split countries by trust level, showing high-trust countries had a case growth rate of 0.0420 compared to 0.0674 for low-trust countries.

This comparison clearly demonstrated that lower trust levels correlated with higher case growth rates and greater stock market volatility during COVID-19. By calculating simple means, researchers transformed thousands of observations into comparable numbers that revealed how high-trust countries maintained more stable markets during the pandemic.

Dataset Activity

Tasks

Using the small COVID-19 dataset, perform the following activities:

  1. Calculate and interpret measures of central tendency and dispersion for COVID-19 cases across Indian states during January-March 2020.
  2. Create appropriate visualizations:
    • Histograms with density curves for each variable
    • Boxplots to identify outliers
    • Violin plots to show distribution shape

R Code

Below is my R code implementation for this activity:

# ============================================
# COVID-19 India Descriptive Statistics
# ============================================

# Step 1: Load required libraries
library(readxl)
library(ggplot2)

# Step 2: Load the dataset
df <- read_excel("Covid19 India (Jan 20 - Mar 20).csv.xls")

# Step 3: Create a function to calculate summary statistics
summary_stats <- function(x) {
  c(
    Mean = mean(x, na.rm = TRUE),
    Median = median(x, na.rm = TRUE),
    SD = sd(x, na.rm = TRUE),
    Variance = var(x, na.rm = TRUE),
    Min = min(x, na.rm = TRUE),
    Max = max(x, na.rm = TRUE)
  )
}

# Step 4: Calculate descriptive statistics for all variables
summary_stats(df$ConfirmedIndianNational)
summary_stats(df$ConfirmedForeignNational)
summary_stats(df$Cured)
summary_stats(df$Deaths)

# ============================================
# Step 5: Create Histograms with Density Curves
# ============================================

# Histogram 1: Indian Nationals
hist(df$ConfirmedIndianNational, probability = TRUE,
     main = "Histogram: Indian Nationals",
     xlab = "Confirmed Indian Nationals")
lines(density(df$ConfirmedIndianNational, na.rm = TRUE), col = "red", lwd = 2)

# Histogram 2: Foreign Nationals
hist(df$ConfirmedForeignNational, probability = TRUE,
     main = "Histogram: Foreign Nationals",
     xlab = "Confirmed Foreign Nationals")
lines(density(df$ConfirmedForeignNational, na.rm = TRUE), col = "red", lwd = 2)

# Histogram 3: Cured Cases
hist(df$Cured, probability = TRUE,
     main = "Histogram: Cured Cases",
     xlab = "Cured Cases")
lines(density(df$Cured, na.rm = TRUE), col = "red", lwd = 2)

# Histogram 4: Deaths
hist(df$Deaths, probability = TRUE,
     main = "Histogram: Deaths",
     xlab = "Deaths")
lines(density(df$Deaths, na.rm = TRUE), col = "red", lwd = 2)

# ============================================
# Step 6: Create Boxplot to Identify Outliers
# ============================================

boxplot(df$ConfirmedIndianNational,
        df$ConfirmedForeignNational,
        df$Cured,
        df$Deaths,
        names = c("Ind Nat", "For Nat", "Cured", "Deaths"),
        main = "Boxplots of COVID-19 Variables",
        ylab = "Values")

# ============================================
# Step 7: Create Violin Plots
# ============================================

# Violin Plot 1: Indian Nationals
ggplot(df, aes(x = "", y = ConfirmedIndianNational)) +
  geom_violin(fill = "lightblue") +
  ggtitle("Violin Plot: Indian Nationals")

# Violin Plot 2: Foreign Nationals
ggplot(df, aes(x = "", y = ConfirmedForeignNational)) +
  geom_violin(fill = "lightgreen") +
  ggtitle("Violin Plot: Foreign Nationals")

# Violin Plot 3: Cured Cases
ggplot(df, aes(x = "", y = Cured)) +
  geom_violin(fill = "orange") +
  ggtitle("Violin Plot: Cured Cases")

# Violin Plot 4: Deaths
ggplot(df, aes(x = "", y = Deaths)) +
  geom_violin(fill = "pink") +
  ggtitle("Violin Plot: Deaths")
