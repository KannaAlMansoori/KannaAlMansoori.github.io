Dataset

COVID-19 India Cases (January 2020 - March 2020)

Source: This dataset contains early COVID-19 case data from India covering the initial months of the pandemic from January to March 2020. The data includes confirmed cases (both Indian and foreign nationals), recovered cases, and deaths across different states and union territories of India.

Reflection

This activity provided me with a solid foundation in exploratory data analysis using R. The main challenge I encountered was formatting the date column correctly to extract the date range. Initially, the dates were not in the proper format, which prevented me from using functions like min() and max() effectively.

To overcome this, I had to first identify and remove any NA values in the Date column, then convert the dates using the as.Date() function with the appropriate format specification. Once the dates were properly formatted, I was able to successfully determine the date range covered by the dataset (January 2020 to March 2020).

This experience taught me the importance of data cleaning and proper data type conversion before performing any analysis. It also reinforced the need to carefully examine data structures and handle missing values appropriately in real-world datasets.
