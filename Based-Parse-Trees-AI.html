Constituency-Based Parse Trees

Natural Language Parsing Activity

Type: Formative Activity | Status: ✓ Complete

Activity Guidance and Learning Outcomes

Activity Guidance

Create a constituency-based parse tree for each of the following phrases and add the completed trees to the e-portfolio:

  • The government raised interest rates.
  • The internet gives everyone a voice.
  • The man saw the dog with the telescope.

Intended Learning Outcomes

  • Develop the knowledge and skills required to develop, deploy and evaluate the tools and techniques of intelligent systems to solve real-world problems
  • Develop an understanding of contemporary research issues in the area of intelligent agent systems

Method: Constituency Parsing

Constituency parsing breaks a sentence into nested constituents — phrases that behave as single grammatical units — using phrase-structure grammar. The core rule is S → NP VP: every sentence splits into a subject noun phrase and a predicate verb phrase, which are then broken down further until every word is tagged.

Labels used: S = sentence · NP = noun phrase · VP = verb phrase · PP = prepositional phrase · Det = determiner · N = noun · V = verb · P = preposition · Pron = pronoun

Parse Trees

1. The government raised interest rates.

Figure 1: Constituency parse tree for "The government raised interest rates."

Summary: A straightforward S → NP VP structure. The subject NP is "the government"; the predicate VP is "raised interest rates". "interest rates" is a compound noun, so "interest" sits inside the object NP as a noun modifier of "rates".

2. The internet gives everyone a voice.

Figure 2: Constituency parse tree for "The internet gives everyone a voice."

Summary: The verb "gives" is ditransitive — it takes two objects — so the VP branches into two noun phrases: the indirect object "everyone" and the direct object "a voice". This illustrates how the structure of a verb phrase depends on the verb's argument requirements.

3. The man saw the dog with the telescope.

This sentence is structurally ambiguous — the prepositional phrase "with the telescope" can attach in two different places, giving two valid trees and two meanings.

Reading A — the telescope is the instrument (the man used it to see). The PP attaches to the VP:

Figure 3: Reading A — the PP "with the telescope" attaches to the VP (instrument reading).

Reading B — the dog has the telescope. The PP attaches to the NP "the dog":

Figure 4: Reading B — the PP "with the telescope" attaches to the NP "the dog".

Summary: This is a classic case of PP-attachment ambiguity. Grammar alone cannot decide which reading is correct — both are structurally valid. This is exactly the kind of ambiguity that intelligent systems performing natural language understanding must resolve, typically by combining grammatical rules with context or with statistical/probabilistic parsing.

Link to the Learning Outcomes

Tools and Techniques of Intelligent Systems

Constituency parsing is a foundational technique in natural language processing — the component that lets an intelligent agent recover grammatical structure from raw text before it can interpret meaning. Producing these trees manually demonstrates the underlying method that automated parsers (e.g. the Stanford Parser, NLTK, spaCy) implement at scale.

Contemporary Research Issues in Intelligent Agent Systems

Sentence 3 shows that structural ambiguity is a problem that grammar alone cannot solve. Resolving PP-attachment and similar ambiguities remains an active area of NLP research, and it is central to how conversational and language-understanding agents interpret human input correctly. This connects the exercise directly to the challenges facing modern intelligent agent systems.
